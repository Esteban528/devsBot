# devsBot

## Instalación (Es necesario tener ssh vinculado y funcionando)
```bash
git clone git@github.com:Esteban528/devsBot.git

cd devsBot
npm i
```

## Ejecución 
```bash 
node .
```

## Subir cambios
```bash
git add <nombre del archivo>
git commit -m "Mensaje de la actualización/commit"
git push origin main

```
